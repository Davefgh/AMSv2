USER INTERACTION FLOW:

1. User opens Schedules screen
   ↓
   "Today" filter selected by default
   ↓
   Shows today's schedules

2. User clicks "Monday" chip
   ↓
   State updates to "Monday"
   ↓
   Schedules filter to Monday only
   ↓
   Header updates to "Schedules for Monday"

3. User clicks "Friday" chip
   ↓
   State updates to "Friday"
   ↓
   Schedules filter to Friday only
   ↓
   Header updates to "Schedules for Friday"

4. If no schedules for selected day
   ↓
   Empty state displays
   ↓
   User can select different day
